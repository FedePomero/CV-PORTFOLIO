Se diseño estructura con html cv-portafolio web
Se utilizaron herramientas css para diseño (animaciones, keyframes, etc)
Se utilizo carrusel de bootstrap para definir la seccion de testimonios dentro del cv-portafolio.
Se utilizo mediaquery para que el sitio sea responsivo. 
Se utilizaron fuentes de letras de Google Fonts.
Se utilizaron iconos de Bootstrap.
Se utilizo una funcion de javascript para mostrar / ocultar menu en la barra de navegacion y para darle funcion al boton "ir arriba".
